# MLHW1
从以下主题中选择一项，利用AI编程工具辅助设计对应的软件前端原型。
注意：该题可小组组队完成，团队成员不超过4人，也可单人完成。若超过1人完成，请注明每个人的具体工作内容和可量化的百分比，若工作内容和百分比不注明，统一视为0分。
（2）主题二：面向城市交通治理的低空无人机智能分析系统
用户角色：操作员、管理员
功能：用户登录、资源地图展示（地图加载展示、拍摄点位地图展示、数据类型展示）、视频数据管理（按任务视频列表、视频播放、查看视频属性、视频抽帧、查看抽帧照片、下载视频、删除视频）、作业态势监控（提供无人机\基站作业过程全态势监控能力，包括对无人机作业进行实时监控，实时获取无人机直播画面、位置、朝向、任务完成进度、电池、飞行状态等信息）、计划作业（提供无人机/基站定时调度作业功能，包括支持无人机计划调度作业，可预先规划任意时间点周期或非周期巡查任务，无人机到点根据天气条件、设备状态情况自动作业）、航线设定（提供新建航线任务、航线信息查询、航线信息录入、设置起降航高、设置航线方向、高度、设定航线飞行速度、航点自由设置（位置、悬停时间、动作、云台俯仰角、飞行器偏航角模式）、设置往返速度、航线设定实时显示等功能）、AI模型库管理（提供模型库列表、模型库详细信息显示、模型部署等功能）、系统设置（用户角色管理、运维管理）
