# MLHW1
从以下主题中选择一项，利用AI编程工具辅助设计对应的软件前端原型。
注意：该题可小组组队完成，团队成员不超过4人，也可单人完成。若超过1人完成，请注明每个人的具体工作内容和可量化的百分比，若工作内容和百分比不注明，统一视为0分。
（1）主题一：数字病理教学平台
用户角色：学生、教师、管理员
功能：用户登录、病例资料（含pdf、ppt、视频、.svs切片文件）上传、教学课件展示（类似MOOC）、考题录入、学生考试、智能组卷、试卷批改、讨论交流、SVS切片浏览与处理（标注、测量、计数、导航）、切片管理、学生进度可视化、RAG引导的知识点或者考试题分析、系统设置（支持前述各子功能进行是否启用、用户管理、系统日志）
